# Changelog

## v0.2.0-b2 — 2026-07-24

- Зафиксирован единый канонический snapshot Version B2.
- Интегрированы frontend, Express/Prisma backend и Telegram consumer.
- Добавлены JWT-сессии, восстановление авторизации и API persistence поездок.
- Добавлены маршрутные изменения, история событий и Telegram-уведомления.
- Добавлен AI assistant-context для маршрута и погоды.
- Добавлены генерация трёх Plan B, проверка и применение выбранного плана.
- Добавлены документы, OCR, приглашения, участники, SOS и monitoring API.
- Добавлены безопасные `.env.example`, CI, secret scan и эксплуатационная
  документация.

Version B2 остаётся демонстрационным стендом: реальные GDS-бронирования и
платёжные операции не выполняются.
