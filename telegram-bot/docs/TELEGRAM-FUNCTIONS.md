# Функции Telegram-бота

## Привязка

`/start` без аргумента показывает состояние аккаунта. Deep link `/start link_<token>` передаёт одноразовый token в `TravelApiClient`; JWT сайта не используется. Полные tokens не логируются. `/unlink` удаляет Telegram-привязку и активную поездку.

Mock tokens: `link_demo-artem`, `link_demo-anna`.

## Поездки и события

- `/trips` - активные, планируемые поездки и приглашения с ролью;
- `/history` - завершённые поездки только для просмотра;
- `/today` - события локального дня активной поездки;
- `/next` - ближайшее событие;
- выбор поездки сохраняется в техническом состоянии/backend;
- чужие, отозванные и непринимаемые приглашения не открываются.

## Документы и сообщения

- `/documents` показывает только разрешённые документы;
- mock PDF отправляется как Telegram document через `FSInputFile`;
- API mode выдаёт временную backend-ссылку;
- `/messages` показывает опубликованные сообщения организатора и Plan B;
- кнопки сайта открывают workspace поездки, вкладка выбирается вручную.

## SOS

`/sos`: поездка → сегмент/вся поездка → категория → непустое описание → предпросмотр → подтверждение. Есть `/cancel`, кнопки отмены, idempotency key, защита FSM от двойного клика и rate limit. `/mysos` показывает только собственные обращения.

Организатор получает уведомление. Подтверждение нарушения и выбор Plan B выполняются только на сайте.

## Уведомления

Поддерживаются напоминания о сегментах, время/выход/терминал/платформа/место отправления, задержка/отмена, трансфер/отель, документы, приглашения, SOS, нарушение, Plan B и сообщения. Dispatcher учитывает preferences, quiet hours, deduplication, delivery ack и bounded retry временных ошибок.

`/notifications` меняет категории, тихие часы и timezone. `/demo` добавляет демонстрационные события только в development+mock.

## AI

`/assistant` отвечает только по разрешённому контексту поездки. Он не изменяет поездку, не отправляет SOS, не подтверждает нарушение, не выбирает Plan B и не удаляет данные. `/cancel` и кнопка завершения выходят из FSM.
